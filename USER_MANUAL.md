# 卷王圈 (YuePaoQuan) 跑步助手用户手册

欢迎进入**卷王圈**！这是一个基于大语言模型（VLM/LLM）的智能跑步激励和里程追踪系统。我们将无情地鞭策每一位跑者，告别咸鱼，向极限进发！

## 🤖 核心功能概览

系统主要由两个部分组成，日常请直接与私信助理 App（秘书Jack/卷王助理等）互动，一切高光（或划水）表现都将被全群同步：

1. **私聊助理 (App)**：你设目标、传图片、查进度的唯一指定接收处。
2. **群聊广播 (Webhook)**：全群高音喇叭，向全群公开你的战报和图片！

---

## 💬 核心指令指南

在企业微信中搜索或者钉在你的聊天列表里的「应用助理」，发送以下自然语言，助理会自动识别并更新你的状态：

### 1. 🎯 立下军令状（设定月度目标）
每个月的第一步，请务必喊出你的跑量目标。  
**发送示例：**
- `我这月的目标是 200`
- `目标300公里`
- `本月跑量目标为 150K`

💡 **系统动作**：助理会私聊回复你「立下军令状」，同时去群里**大广播**你的宣战誓言。

### 2. 🏃‍♂️ 打卡记录跑量（发送配速截图）
每次跑完，只需截图你的跑表/App（佳明、高驰、华为、Keep、Strava等，需包含配速和距离等数据），**作为图片发送给助理**。
无需多说废话，助理的视觉 AI（Vision OCR）小雷达会瞬间启动：

💡 **系统动作**：
1. 智能抓取单次：**距离、配速、心率、耗时等数据**。
2. **毒舌卷王辣评**：根据你的配速和实力段位，生成属于你的专属点评。
3. **群内高光转播**：截图原图、文本化战绩和 AI 辣评**会无情地被发送到群聊**，炫耀或者被群嘲，就看你配速了！

### 3. 📈 查询个人进度
不知道这个月还差多少？随时跟助理对线。  
**发送示例：**
- `到目前我的进展如何`
- `本月进度`
- `查询`
- `跑了多少？`

💡 **系统动作**：私聊回复你的战报（已跑公里数、任务进度%、剩余公里数），同时也会广撒网把你的个人战报放到群里去。

### 4. 🏆 查看卷王排行榜
想知道其他人在偷偷卷了多少？查看当月的“修罗场”战力榜吧！    
**发送示例：**
- `排行榜`
- `卷王榜`
- `跑量榜`
- `榜单`

💡 **系统动作**：助理私下并向群内公布当月所有人的**累计里程排行榜**（降序），以及每个人当前达标的情况！

## 🛠 给管理员的运维指南 (GCP 服务器)

如果后台服务需要更新或维护，相关命令如下：

1. 登录 GCP VM 进入目录：
   ```bash
   cd ~/juanwang
   ```
2. 拉取最新代码并重启服务：
   ```bash
   git pull
   pm2 restart yuepaoquan
   ```
3. 查看实时运行日志（监控 OCR 是否成功，或是报错排查）：
   ```bash
   pm2 log yuepaoquan --lines 20
   ```
4. 如果缺失包依赖，需切换至虚拟环境后安装：
   ```bash
   source venv/bin/activate
   pip install -r requirements_server.txt
   ```
